## Build Setup

npm
``` bash
# install dependencies
npm install

# serve with hot reload at 0.0.0.0
npm run start

# build for production with minification
npm run build

```
or using yarn
``` bash
# install dependencies
yarn install

# serve with hot reload at 0.0.0.0
yarn start

# build for production with minification
yarn build

```
